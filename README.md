# a simple MERN stack chat web app

### Project Setup

While in the frontend directory and backend directory

```sh

npm install

```

### Compile and Hot-Reload for Development

While in the frontend directory

```sh

npm run dev

```

While in the backend directory

```sh

nodemon ./server.js

```
